CWatM Model

class CWatM_Model(start_date, n_steps, settings)[source]

This class is used to initalize the CWatM model from GEB. Several static configuration files are read first, then several dynamic parameters are set based on the configuration of GEB. Then, the model frame is created that can then later be used to iteratate.

Parameters

  • start_date (datetime) – Start date of the model.

  • n_steps (int) – Number of steps that the model will run for.

  • settings (str) – Filepath of the CWatM settingsfile. For full configuration options please refer to the CWatM documentation <https://cwatm.iiasa.ac.at/>.

  • use_gpu – Whether the model can use a GPU.

Init part of the initial part defines the mask map and the outlet points initialization of the hydrological modules
